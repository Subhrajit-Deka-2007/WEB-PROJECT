# WEB-PROJECT

A collection of web development projects built with HTML, CSS, and JavaScript.

## 📂 Project Structure

This repository contains multiple web projects showcasing different web development concepts and features.

### Project1: Tip Calculator

A simple and interactive web application for calculating tips on restaurant bills.

**Features:**
- Enter bill amount
- Specify tip percentage
- Automatic calculation of total amount including tip
- Clean and user-friendly interface

**Technologies Used:**
- **HTML** - Semantic markup for structure
- **CSS** - Styling and responsive design
- **JavaScript** - Interactive functionality

**How to Use:**
1. Enter the bill amount in the "Bill Amount" field
2. Enter the tip percentage in the "Tip Percentage" field
3. Click the "Calculate" button
4. The total amount (bill + tip) will be displayed in red

**File Structure:**
```
Project1/
├── index.html    # Main HTML structure
├── style.css     # Styling and layout
└── script.js     # JavaScript functionality
```
